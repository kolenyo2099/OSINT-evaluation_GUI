This tool allows researchers to automatically search for and scrape X/Twitter threads hosted on Threadreaderapp.com that in turn are sent to OpenAI's GPT for evaluation based on custom instructions. The tool makes use of the following procedures:

  • Run main script with username or Threadreader URL as input (detailed below)
  • Search for threads of this user on Threadreader in Programmable Google Search Engine
  • Scrape and save thread URLs
  • Extract posts from threads
  • Evaluate posts per thread
  • Dump result
